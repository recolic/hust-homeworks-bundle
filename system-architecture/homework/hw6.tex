\smallskip

没有远程访问时, $BaseCPI = 1$. \\

有远程访问时, $CPI = BaseCPI + ExtraCost$, 即

$$CPI = 1 + 0.5\% \frac{2000}{10} = 2$$

因此, 前者的程序运行速度是后者的2倍.
